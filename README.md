# SceneAI — AI Assistant for Godot 4

AI plugin that reads your scene and explains 
why things are broken.

## Installation
1. Download all files
2. Put in your project: addons/godot_ai/
3. Enable in Project → Plugins → SceneAI

## Setup
1. Choose your AI provider (Groq is free)
2. Get API key from your provider
3. Paste key in the SceneAI panel
4. Ask anything about your scene

## Supported Providers
- Groq (free)
- Google Gemini
- Anthropic Claude
- OpenAI GPT-4o mini
- OpenRouter
- Mistral AI
- Cohere

## Usage
Open any scene → type your problem → 
click "Ask AI" → get instant answer
